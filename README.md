# Personal Portfolio Website

A modern, responsive personal portfolio website built with React and Tailwind CSS, featuring a luxury design with gold accents and smooth animations.

![Portfolio Screenshot](public/img/portfolio-screenshot.png)

## 🌟 Features

- **Responsive Design**: Optimized for all device sizes from mobile to desktop
- **Theme Toggling**: Dark and light mode support with elegant transitions
- **Interactive UI Elements**: Hover effects, animations, and micro-interactions
- **Modern Stack**: Built with React, TypeScript, and Tailwind CSS
- **Optimized Performance**: Fast loading and smooth scrolling
- **Luxury Aesthetics**: Premium design with attention to detail
- **Animated Sections**: Elements animate as you scroll through the page
- **Project Showcase**: Highlight your work with interactive project cards

## 🚀 Technologies Used

- **React**: Frontend library for building the user interface
- **TypeScript**: Type safety for more reliable code
- **Tailwind CSS**: Utility-first CSS framework for styling
- **React Router**: For handling navigation
- **Lucide Icons**: Beautiful, consistent icon set
- **React Query**: For efficient data fetching
- **Intersection Observer API**: For scroll-based animations

### Styling

The project uses Tailwind CSS for styling. Custom styles and theme variables.
